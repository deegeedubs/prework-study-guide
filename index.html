HTML

  • Enter !DOCTYPE html to tell the browser that this is an HTML document.
  • The html element will signify the beginning of the HTML code.
  • lang="en" will specify that we are working in the english language.
  • The head element contains info about the webpage.
    • Title will define the title of the webpage.
    • Base will define the base URL for the site.
    • Link will connect an external resource to the HTML file.
    • Meta will define the Metadata, such as character set, description, keywords, author, and viewport.
    • Script will embed the client side scripts into the HTML code.
  • The body element contains the visible content shown to the user.
    • The ul element indicates an unordered list, while the ol element indicates an ordered list.
    • List elements are denoted with li.
    • Headings are denoted with h1, h2, h3, etc. with h1 being the largest.
    • The img element will inicate the use of an image.
      • Use src=" " to define the location of the image and alt=" " to give a description of the image.
    • The br element will enter a line break
  • The section element is used to define a section of code for thematic content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status : checks what branch we are working in and whether it is up to date.
  • git checkout : navigates to a defined branch./li>
    • git checkout -b branch-name : creates a new branch and places us there.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
